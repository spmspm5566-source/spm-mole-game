📢 使用說明
首次設定步驟：
1. 「系統設定」填入學校資料、科任班級、導師班級
2. 「學生名單」匯入 CSV 或手動新增（含家長1/家長2 LINE ID）
3. 「LINE 設定」填入 Cloudflare Worker 網址與 Channel Access Token
4. 「學年度管理」確認目前學年，每年7月執行升班
5. 各子系統開始使用

分享給其他老師：每位老師自己的 GitHub Pages 網址是獨立的，資料互不干擾。
篩選班級
📂 匯入學生名單 CSV

student_id, class_name, seat_number, student_name, gender, student_line_id, parent1_line_id, parent1_name, parent1_relation, parent2_line_id, parent2_name, parent2_relation

📅 學年度升班（每年7月執行）
